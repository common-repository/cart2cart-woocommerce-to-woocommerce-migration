1. Log in your WordPress admin panel
1. Go to Plugins > Add New > Browse. Pick Cart2Cart plugin and click “Install Now” button. 
1. Activate it by clicking “Activate plugin”.
1. Find Cart2Cart plugin on the left side menu. Register Cart2Cart account and proceed with the migration setup.

== Frequently Asked Questions ==

= Is it possible to keep the current WooCommerce SEO rankings on the new WooCommerce store? =
Absolutely. You only have to choose the Migrate 301 redirects option on the corresponding Migration Wizard step.

= Is taxes migration supported by the plugin? =
Sure. But keep in mind that after migration to the new WooCommerce store, the taxes will be already included in product prices.

= Can I preserve order IDs whilst migrating to WooCommerce? =
Yes, WooCommerce to WooCommerce service by Cart2Cart makes it possible to preserve order IDs on the new WooCommerce store. However, in order to do that successfully, you need to install the free [WooCommerce Sequential Order Number](https://wordpress.org/plugins/woocommerce-sequential-order-numbers/) module and choose the Preserve order IDs on the Target Store option on the appropriate Migration Wizard step.

= How to migrate manufacturers WooCommerce via the plugin? =
To enable the migration of manufacturers to your new WooCommerce store, you should install the [WooCommerce Brands](https://woocommerce.com/products/brands/) plugin (it's paid).

= Is there a possibility to migrate customer passwords? =
Yes, of course. With the WooCommerce data migration tool by Cart2Cart, you can transfer your WooCommerce store customers’ passwords, provided that the store has default settings and no custom code modifications have been made.
